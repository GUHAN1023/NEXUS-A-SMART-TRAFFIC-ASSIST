Setup Guide
Follow these steps to make the app fully real with Firebase + Real Maps. All free. No cost for small usage.
1
Create Firebase Project (Free)
Go to console.firebase.google.com
→ Create a project → Add a Web App
→ Copy the config object shown
// firebase config you get const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "yourapp.firebaseapp.com", projectId: "yourapp-12345", storageBucket: "yourapp.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc123" };
2
Enable Firebase Services
In Firebase Console, enable:
→ Authentication → Anonymous sign-in ✓
→ Firestore Database → Create database
→ Start in test mode (for development)
3
Set Firestore Security Rules
In Firestore → Rules tab, paste this:
rules_version = '2'; service cloud.firestore { match /databases/{db}/documents { // Anyone can read reports match /traffic_reports/{doc} { allow read: if true; // Only authenticated users can write allow write: if request.auth != null; } // Emergencies — read only for public match /emergencies/{doc} { allow read: if true; allow write: if request.auth != null; } } }
4
Paste Your Config Into This File
In this HTML file, find the firebaseConfig object (near top of <script type="module">) and replace the placeholder values with your real config from Step 1.
// Replace these placeholders: apiKey: "YOUR_API_KEY", ← paste real projectId: "YOUR_PROJECT_ID", ← paste real // ... etc
5
Real Traffic Data (Optional Upgrade)
For live congestion data, sign up free at:
→ developer.tomtom.com (2,500 calls/day free)
→ Replace the mock API values with:
// Real TomTom traffic API call const ERODE = '11.3410,77.7172'; const res = await fetch( `https://api.tomtom.com/traffic/services/4/` + `flowSegmentData/absolute/10/json?` + `point=${ERODE}&key=YOUR_TOMTOM_KEY` ); const d = await res.json(); const speed = d.flowSegmentData.currentSpeed; const freeFlow = d.flowSegmentData.freeFlowSpeed; const congestion = 1 - (speed / freeFlow); setCongestion(Math.round(congestion * 100));
6
Make It Installable (PWA)
Add these files alongside your HTML to make it installable on any phone — no app store needed:
// manifest.json { "name": "NEXUS Traffic", "short_name": "NEXUS", "start_url": "/index.html", "display": "standalone", "background_color": "#0d1421", "theme_color": "#00c8ff", "icons": [{ "src": "icon-192.png", "sizes": "192x192" }] } // In your HTML <head>: <link rel="manifest" href="manifest.json"> <meta name="theme-color" content="#0d1421">
7
Host It Free (Firebase Hosting)
Install Firebase CLI and deploy:
# In your terminal: npm install -g firebase-tools firebase login firebase init hosting firebase deploy # Your app is now live at: # https://yourapp.web.app ✅
