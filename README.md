NEXUS — Smart City Traffic Intelligence System

A futuristic, real-time traffic intelligence platform designed to optimize urban mobility using AI-driven insights, live data streams, and smart city infrastructure.

NEXUS is a full-stack smart traffic management system that integrates:

 Live GPS tracking
 AI-based signal optimization
 Smart route recommendations
 Emergency vehicle prioritization
 Intelligent parking management
 Real-time analytics dashboard

FRONTEND
 HTML5, CSS3, JavaScript
Canvas API (heatmaps, charts)
Leaflet.js (maps)
Responsive UI design
